Basic HTML code

  • First
    You have to include css and javascript MixSlide files (mixSlide.css & mixSlide.js)
  • Second
    You must have a division with a particular html code. That division must have an identifier
    • Differents divs tag in the parent div
      According to the number of the number of images
    • One img tag in each div
    • One p tag in each div to give a label to the image. It must be a description or a imple name. This tag is optional

CSS Style

You can change the style of some object on the slider. Here is the full list of changeable objects (selectors)

  • Overlay in fullscreen mode
    .mixSlide-frame.fullscreen : background (default : rgba(0,0,0,.7))
  • Container background
    .mixSlide-container : background (default : black)
  • Labels for images
    .mixSlide-images p : all properties to stylize text, background, color, font, etc.
